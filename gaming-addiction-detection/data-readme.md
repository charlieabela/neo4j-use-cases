# Gaming Addiction Detection Dataset

This dataset contains information about 50 players across 6 games, designed to demonstrate graph analytics for behavioral analysis.

## Files in this dataset:

- **players.csv** - 50 player profiles with demographics
- **games.csv** - 6 games with addictiveness scores
- **sessions.csv** - 847 gaming sessions with timestamps and duration
- **friendships.csv** - Social network connections between players
- **purchases.csv** - In-game purchase transactions

These files are designed to be loaded into Neo4j for graph database analysis. See the main repository README for loading instructions.
